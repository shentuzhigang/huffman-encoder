# 赫夫曼编码-译码器
## 基本概念
哈夫曼编码(Huffman Coding)：又称霍夫曼编码、赫夫曼编码-，是一种编码方式，哈夫曼编码是可变字长编码(VLC)的一种。Huffman于1952年提出一种编码方法，该方法完全依据字符出现概率来构造异字头的平均长度最短的码字，有时称之为最佳编码，一般就叫做Huffman编码（有时也称为霍夫曼编码）。
